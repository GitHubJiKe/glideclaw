## 🚀 使用示例

### 启动管理界面
```bash
export API_KEY="your-api-key"
bun run web
# 访问 http://localhost:8001
```

### 在代码中使用
```typescript
import { AssistantManager } from "./src/core/assistant";
import { loadConfig } from "./src/core/config";
import { MetaStore } from "./src/core/meta";

const config = await loadConfig();
const meta = new MetaStore({ dbPath: config.dbPath });
const manager = new AssistantManager(meta, config);

// 获取系统提示词
const prompt = manager.getSystemPrompt("glideclaw-default");

// 创建新助手
const { agentId } = manager.createAssistant("分析助手", "数据分析");

// 启动定时任务
await manager.initialize();
```

## 📦 部署建议

### 环境要求
- Bun >= 1.3.0
- Node.js (可选，仅作为参考)
- SQLite 3 (内置)
- 现代浏览器 (ES6+)

### 部署步骤
1. `bun install` - 安装依赖
2. 配置 `.glideclawrc` 或环境变量
3. `bun run build` - 构建项目
4. `bun run web` - 启动服务

### 生产建议
- 使用反向代理（nginx）
- 启用 HTTPS
- 定期备份数据库
- 监控服务运行状态
- 设置日志收集
